Правила именования

  • имена переменных должны быть существительными (id, name, count);
  • имена функций должны начинаться с глаголов (getId, parseName, setCount);
  • имя константы должно быть записано прописными буквами, а слова должны быть разделены нижним подчеркиванием (MAX_COUNT);
  • имя объекта, обернутого jQuery, должно начинаться с $ ($el, $this);
  • имена функции-конструкторов должны начинаться с заглавной буквы (Plugin, Number, String);
  • избегайте слишком абстрактных названий, например, temp, foo, bar.

Strict Mode / Строгий режим

"use strict" — это директива, введенная стандартом ECMAScript 5. Эта директива говорит, что следущий за ней программный код выполняется в строгом режиме. Действует в рамках области видимости, в которой объявлена.

Строгий режим определяет ограниченное подмножество языка, благодаря чему исправляет некоторые недостатки языка, а также обеспечивает более строгую проверку на наличие ошибок и повышенный уровень безопасности.

Strict Mode. Ограничения.

  • запрещена конструкция with;
  • все переменные должны объявляться через var;
  • функции, которые вызываются как функции (а не методы), получают в ссылке this значение undefined;
  • попытка определить в литерале объекта два свойства с одинаковыми именами вызовет синтаксическую ошибку;
  • попытка определить два и более параметра с одинаковыми именами в объявлении функции вызовет синтаксическую ошибку;

Strict Mode. Ограничения.

  • не допускается использование литералов восьмеричных целых чисел;
  • идентификаторы eval и arguments интерпретируются как ключевые слова и нельзя переопределить их значения;
  • ограничивается возможность просмотра стека вызовов. Обращение к свойствам arguments.caller и arguments.callee возбуждает исключение TypeError;
  • объект arguments в функции хранит статическую копию значений, переданных фукнции;
  • и другое ...

JavaScript is Evil

Область видимости

Новая область видимости создается только для функций, а не для блоков { }.

Замыкания

IIFE — Immediately Invoked Function Expression

			(function($, window, document, undefined) {
			    'use strict';
			    ...
			})(jQuery, this, document);

Образует изолированную область видимости.

Глобальная область видимости

  1. Конфликты переменных в вашем коде
  2. Конфликты ваших переменных и переменных в подключаемых библиотеках
  3. Конфликты ваших переменных и переменных в аддонах браузера

Ваша переменная может быть попросту переопределена.
